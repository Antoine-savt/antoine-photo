# Portfolio Photographe

Site web portfolio pour photographe professionnel, présentant des galeries de photos, une section à propos et un formulaire de contact.

## Structure du projet

```
portfolio/
├── index.html          # Page principale
├── css/
│   └── styles.css     # Styles CSS
├── js/
│   └── main.js        # Scripts JavaScript
├── images/            # Dossier pour vos images (à créer)
└── README.md          # Documentation
```

## Fonctionnalités

- **Navigation fluide** : Scroll automatique vers les sections
- **Galerie interactive** : Clic sur les images pour les voir en grand format
- **Formulaire de contact** : Formulaire avec validation
- **Design responsive** : Adapté aux mobiles, tablettes et ordinateurs
- **Animations** : Effets au scroll et transitions fluides

## Utilisation

1. Ouvrez `index.html` dans votre navigateur
2. Personnalisez le contenu dans `index.html`
3. Modifiez les styles dans `css/styles.css`
4. Ajoutez vos propres images dans le dossier `images/`

## Personnalisation

### Changer les images
Remplacez les URLs Unsplash dans `index.html` par vos propres images :
- Placez vos images dans le dossier `images/`
- Mettez à jour les chemins dans la section portfolio

### Modifier les couleurs
Éditez les variables de couleur dans `css/styles.css` :
- `#222` : Couleur principale (header, footer, boutons)
- `#ffc107` : Couleur d'accent (hover, liens)
- `#f4f4f4` : Couleur de fond

### Ajouter des sections
1. Ajoutez une nouvelle section dans `index.html`
2. Ajoutez un lien dans la navigation
3. Personnalisez les styles si nécessaire
